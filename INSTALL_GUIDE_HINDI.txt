============================================
   ALGO TRADER - INSTALLATION GUIDE
   (Hindi Instructions)
============================================

STEP 1: Python Install Karo
---------------------------
1. Ye link kholo: https://www.python.org/downloads/
2. "Download Python" button pe click karo
3. Downloaded file pe double-click karo
4. IMPORTANT: "Add Python to PATH" checkbox ZAROOR check karo
5. "Install Now" pe click karo
6. Installation complete hone do


STEP 2: Algo Trader Download Karo
---------------------------------
1. Ye link kholo: https://github.com/anirudhatalmale6-alt/algo-trader-phase1
2. Green "Code" button pe click karo
3. "Download ZIP" pe click karo
4. ZIP file download hogi
5. ZIP file pe right-click karo -> "Extract All" karo
6. Koi bhi folder mein extract kar do (e.g., Desktop)


STEP 3: Install Karo
--------------------
1. Extracted folder kholo (algo-trader-phase1)
2. "install_windows.bat" file pe DOUBLE-CLICK karo
3. Black window khulegi - installation hogi
4. Jab "INSTALLATION COMPLETE" dikhe, Enter dabao
5. Desktop pe "Algo Trader" shortcut ban jayega


STEP 4: App Chalao
------------------
1. Desktop pe "Algo Trader" pe double-click karo
2. App khul jayegi!


BROKER SETUP (App ke andar):
---------------------------
1. App mein "Settings" tab kholo
2. "Add Broker" pe click karo
3. Upstox ya Alice Blue select karo
4. API Key aur Secret dalo
5. "Get Login URL" pe click karo
6. Browser mein login karo
7. Jo code mile, wo paste karo
8. "Authenticate" pe click karo
9. Done!

7. API Key aur Secret mil jayega - save karo

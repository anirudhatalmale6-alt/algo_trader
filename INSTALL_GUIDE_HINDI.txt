============================================
   MUKESH ALGO - WEB APP
   INSTALLATION GUIDE (Hindi)
============================================

STEP 1: Python Install Karo (agar pehle se nahi hai)
----------------------------------------------------
1. Ye link kholo: https://www.python.org/downloads/
2. "Download Python" button pe click karo
3. Downloaded file pe double-click karo
4. IMPORTANT: "Add Python to PATH" checkbox ZAROOR check karo
5. "Install Now" pe click karo
6. Installation complete hone do


STEP 2: MUKESH ALGO Download Karo
----------------------------------
1. Ye link kholo: https://github.com/anirudhatalmale6-alt/algo_trader
2. Green "Code" button pe click karo
3. "Download ZIP" pe click karo
4. ZIP file download hogi
5. ZIP file pe right-click karo -> "Extract All" karo
6. Koi bhi folder mein extract kar do (e.g., Desktop)


STEP 3: Install Karo
---------------------
1. Extracted folder kholo (algo_trader-main)
2. "install_windows.bat" file pe DOUBLE-CLICK karo
3. Black window khulegi - installation hogi
4. Jab "INSTALLATION COMPLETE" dikhe, Enter dabao
5. Desktop pe "MUKESH ALGO" shortcut ban jayega


STEP 4: App Chalao
-------------------
1. Desktop pe "MUKESH ALGO" pe double-click karo
   (ya extracted folder mein "start_app.bat" pe double-click karo)
2. Black terminal window khulegi (ye band mat karo!)
3. Browser automatically khulega: http://127.0.0.1:5000
4. App ready hai!


STEP 5: Broker Connect Karo (App ke andar)
-------------------------------------------
1. App mein left side hamburger menu (3 lines) pe click karo
2. "Broker" page pe jao
3. "Alice Blue" select karo
4. User ID dalo (e.g., 515175)
5. Secret Key (API Key) dalo
6. "Connect" button dabao
7. Connected!


UPDATE KAISE KARE (naya version download):
-------------------------------------------
1. Purana folder delete karo (ya rename karo)
2. GitHub se naya ZIP download karo (Step 2 repeat karo)
3. Extract karo
4. "install_windows.bat" dobara run karo (Step 3)
5. "start_app.bat" se app chalao (Step 4)


PROBLEM AAYE TOH:
------------------
- Server terminal mein error ka screenshot bhejo
- Browser mein app ka screenshot bhejo
- Main solve kar dunga
